Detailed Description

Components are data structures which store information relating to a specific behaviour(s)

Components are one of the core parts of the ECS, although they are completely defined by the user, exisiting only as void* in the engine itself. The components module contains all user-defined types as well as an enum indexing them all. To see details on accessing components, see Query
