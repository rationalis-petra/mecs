◆ add_entity()

int add_entity ( Template *  template)(void)

creates a new entity in the engine from the provided tempalte

The template contains two arrays: one which contains components for the engine to give the entity, and another detailing what type thet are. The exact implementation of the template function is left to the user, all that matters is that they provice a valid template.

Parameters
[in]templateA function which returns a Template object
Returns
entity_id: The handle to the created entity within the engine. Allows the engine user to access the entity's components

◆ get_component()

void* get_component ( int  id,
int  type 
)

gets the component of a specific type from the provided entity

Will return a pointer to the entity's component of that type if it exists, or a nullptr otherwise if the entity given is not valid, will either experience undefined behaviour (not in debug mode) or output an error to the log and return null. The same will happen for invalid types

Parameters
[in]idThe id of a specific entity in the engine
[in]typeThe (enum) type that you want to get. types can be registered through register_type()
Returns
a pointer to either null or the object you wanted
