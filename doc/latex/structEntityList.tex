\doxysubsection{Detailed Description}
The entity list datatype, which is used to store the output of a query. 

The entity list is essentially an array with two variables\+: the length (len) and array (entities). The entities array stores the integer ids which can be used to access the components of the entity via \mbox{\hyperlink{group__Base_gac4a80c3a238d32f76cfd6a6b5c3c8586}{get\+\_\+component()}};
