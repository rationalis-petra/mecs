\doxysubsection{Detailed Description}
Often, we want to ask the engine for a collection of entities matching a certain criteria, e.\+g. all entities with a transform, or all entites which are less than some distance from a point, etc. This kind of data retreival can be done via a query on the engine. We have multiple functions to support this.
