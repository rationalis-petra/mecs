\doxysubsection{Detailed Description}
A struct for containing intermediate results of a new entity. 

The template struct stores an array of pointers to copmonets the engine uses the array to load the different components into the correct entity slots. The num\+\_\+components variable is set in the new\+\_\+template function to the current number of components in the system.
