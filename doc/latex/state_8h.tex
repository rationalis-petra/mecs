\doxysubsection{Detailed Description}
A file containing the variables for the state of the engine. 

Contains all generic data structures and information about them in a singe to avoid the cost of having dedicated structures. As such, this header file should only be included in engine-\/specific .c files, as the game should not attempt to access these structures directly for safety reasons.
